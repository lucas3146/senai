Comandos do Git Bash para subir arquivos para o repositório remoto

  1. git init
    2. Cria a estrutura inicial do repositório Git no computador local.
  2. git status
    3. Verifica o status das alterações realizadas no repositório.
  3. git add
    4. Adiciona arquivos ao histórico do projeto, na staging.
  4. git commit
    5. Registra/salva a alteração no repositório.
  5. git log
    6. Permite visualizar as alterações feitas.
  6. git show número-do-commit
    7. Permite visualizar informações sobre qualquer commit.
  7. git remote add origin "destino"
    8. Informa a pasta remota.
  8. git remote -v
    9. Permite visualizar o repositório remoto.
  9. git push -u origin master
    10. Publica as alterações no repositório remoto.
  10. git pull
    11. Baixa as alterações no repositório remoto.
  11. git checkout -b tarefa/minha-primeira-branch
    12. Cria uma branch
    O parâmetro "–b" é utilizado para informar que desejamos criar uma branch e que esse ramo ainda não existe.
  12. git pull origin (nome da branch)
    13. Realizar o download das informações fornecidas pela outra branch
  13. git merge (nome da branch)
    14. Fazer a mesclagem com a outra branch
  14. git tag -a v1.0 -m "minha primeira tag"
    15. Criação de tags
  15. git tag
    16. ver as tags
  16. git show v1.0
    17. Para ver informações sobre uma tag específica, como a v1.0
  17. git push origin --tags
    18. Jogar as tags para o repositório remoto
